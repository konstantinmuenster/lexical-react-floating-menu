# ✍️ Rich Text Editor w/ Lexical & React

## Installation

```sh
git clone https://github.com/konstantinmuenster/lexical-rich-text-react-demo.git
cd lexical-rich-text-react-demo
pnpm install # or npm install
pnpm run dev # or npm run dev
```
